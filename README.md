# Learn Forge

Learn Forge is an interactive project-based learning platform. It allows users to browse projects, submit their own, and track progress using Firebase.

## Features

- **Project Gallery**: Browse a wide range of technical projects.
- **Project Submissions**: Submit your own projects to the community.
- **Authentication**: Secure Google Sign-In and email authentication via Firebase.
- **Dark Minimalist UI**: A premium, focused design for learning.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Firebase Authentication, Firestore
- **Styling**: Custom CSS with a dark mode aesthetic

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.
3. (Optional) Set up your own Firebase project and update `firebase-config.js`.
